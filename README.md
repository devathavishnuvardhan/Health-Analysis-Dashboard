# Health-Analysis-Dashboard

## Overview
The Health Analysis Dashboard is an interactive Power BI project developed to analyze and visualize global health data related to disease outbreaks. The dashboard provides insights into confirmed cases, recovered cases, and death cases across different countries and regions using interactive visualizations

This project helps users understand global health trends through data-driven analysis and graphical representations.

## Dataset Information

The dataset used in this project contains:

- Province/State
- Country/Region
- Latitude and Longitude
-  Date
- Confirmed Cases
- Status
- Area

The data is used to track and analyze health statistics across multiple regions over time
## Dashboard Feature
### KPI Cards

KPI Cards

-    Total Cases
-    Confirmed Cases
-    Recovered Cases
-    Death Cases

### Death by Area Analysis
A bar chart showing the regions with the highest number of deaths.
### Global Map Visualization
A map-based visualization representing country-wise death distribution using bubbles.
### Trend Analysis
A line chart showing the increase in death cases over time.
### Pie Chart Representation
-    Displays the proportion of:
-    Confirmed Cases
-    Recovered Cases
-    Death Cases
### ountry-wise Data Table
Shows detailed country-level death statistics.

## Key Insights
-    The dashboard provides a clear overview of global health statistics.
-    China (Hubei), Italy, Iran, and Spain are among the most affected regions.
-    Death cases show a significant increase over time.
-    Confirmed cases represent the largest portion of total cases.
## Purpose of the Project
The objective of this project is to:
-    Analyze global health data efficiently
-    Monitor confirmed, recovered, and death cases
-    Identify highly affected regions
-    Support better decision-making through visual analytics

## Conclusion
The Health Analysis Dashboard combines multiple visualizations to present complex health data in a simple
and understandable format. It helps users analyze trends, compare regions, and gain meaningful insights
from the dataset
